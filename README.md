# Random Image Generator

A tool that actually generates random images, not just picks a random image from a list.

## Variation

```
background
    colors      +
    gradients   +
    patterns    -
    noise       -

foreground
    letters     +
    digits      +
    shapes
        rect    +
        circle  -
        poly    -
        star    -
        arc     -

misc
    scale       -
    rotate      -
```
